# 📖 Deep-Dive Explainers (one per question)

Each file below is a **complete, standalone lesson** for one question. Every
explainer follows the same friendly structure:

1. **What the problem asks** (plain English)
2. **A real-life analogy** 🧸
3. **The logic / trick**
4. **A diagram** (flowchart or tree)
5. **Building the code step by step** (piece by piece)
6. **The complete program** ✅
7. **A full dry run** 🏃 (line-by-line table with a real example)
8. **Common mistakes** ⚠️
9. **Try it yourself** 🎯

> 🆕 Never coded before? Read the [Concepts Primer](../00_concepts_primer.md) first.
